Rubygems Snapshot by rogerleite

Adds snapshot command to gem. This command allow import/export of gems.

Export and Import your gems. Snapshot, copy your current gems in a single file, providing a fast and secure way to import your gems anywhere.

Install

gem install rubygems_snapshot
